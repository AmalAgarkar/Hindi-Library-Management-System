# 📚 Hindi Library Management System

A web-based Library Management System developed using **PHP**, **MySQL**, **JavaScript**, and **Bootstrap 4**. This system provides a digital solution for libraries to manage book inventories, track issued books, and manage user and administrative records efficiently.

## 🚀 Features

* **Admin Panel:**
  * Add, update, and manage book records.
  * Register and manage library users/students.
  * Issue books and track return dates.
  * View overall library statistics and logs.
* **User/Student Portal:**
  * Browse available books in the library.
  * Check account status and current issued books.
* **Responsive UI:** Built with Bootstrap 4 to ensure a smooth and responsive experience across all devices (desktop, tablet, and mobile).
* **Hindi Language Focus:** Tailored to cater to Hindi-speaking users/institutions (as per the project title).

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 4
* **Backend:** PHP
* **Database:** MySQL
* **Icons/Fonts:** FontAwesome / Webfonts
